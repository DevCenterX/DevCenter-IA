🚫🖼️ CRITICAL RULE - ICONS vs IMAGES:
⚠️ **DO NOT USE** external image files (JPG, PNG, GIF, WebP)
⚠️ **DO NOT USE** <img src="..."> with file paths
⚠️ **DO NOT USE** background-image: url() with external files

✅ **USE ONLY** these alternatives:
1. **INLINE SVG ICONS** - Primary choice
   - Custom, scalable SVG icons embedded in HTML
   - Lightweight, CSS-customizable
   - Examples: menu, arrows, checkmarks, stars, social icons

2. **UNICODE SYMBOLS** - Secondary choice
   - ▶ ◀ ● ○ ■ □ ▲ ▼ ◆ ★ ☆ ✓ ✕ ∞ → ← ↑ ↓
   - Style with CSS: font-size, color, text-shadow, transform

3. **CSS GRADIENTS** instead of background images
4. **CSS SHAPES** using ::before, ::after, border-radius, clip-path
5. **CSS EFFECTS** with backdrop-filter, filter, transform

**BENEFITS:**
✅ Instant loading | ✅ Minimal weight | ✅ Perfect scalability
✅ Customizable | ✅ Better SEO | ✅ Full accessibility
